## 📥 Local Development

To run this portfolio locally on your machine, you can use any standard static file server.

### Prerequisites
- Python 3.x (or any static HTTP server like Node.js `http-server`)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Phanindhra-Kalvala17/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Start a local HTTP server:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`.
